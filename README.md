# Comand Line Crawler
Comand Line Crawler takes the basic idea of an endless dungeon crawler and trys to apply it in a comand line, with it all being coded with C++.

The game's code must be compiled first (otherwise you can't play it, duh), so using any sort of C++ code compiler of your choice must be done prior to attempting to play the game.
```bash
Plush note:
The reason why this is is because, when I originally coded the game, I did so on a very old laptop, one which couldn't run programs like Visual Studio Code and such, meaning I had next to no way of ACTUALY compiling said code.
```

The game's controls are pre-backed into the gameplay itself, so there is no real need or worry when it comes to that part.

I hope you enjoy this weird mess of a game as much as I enjoy making it and learning new things from it as I coded it.
